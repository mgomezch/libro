\begin{center}
\textbf{\large Resumen}
\par\end{center}{\large \par}

\vspace{5mm}


\begin{onehalfspace}
El presente proyecto de pasantía consiste en el diseño e implementación
de una aplicación médica diseñada para atender consultas médicas a
través de un navegador Web. La empresa VENETECNOLOGÍA C.A. (filial
del GRUPO VENEMERGENCIA) tiene como objetivo la creación de esta aplicacion
cuyo fin es permitirle a un médico poder atender a sus pacientes,
almacenando la información de cada consulta médica, para así, mantener
un historial completo del paciente. Este historial permitiría enriquecer
la información de los pacientes, lo que puede conllevar al alcance
de diagnósticos más precisos, reinvertir los tiempos de consulta utilizados
para recopilar información que ya se posee, además mejorar y agilizar
los procedimientos que involucra el desarrollo de una consulta.

La idea es construir un conjunto de formularios, que de manera intuitiva
permitan el desarrollo rápido y efectivo de una consulta médica. También,
permitir acceder al médico a información de consultas previas En base
a esta información el médico debe poder registrar consultas, recetar
prescripciones, solicitar citas de laboratorio o un traslado de ambulancia
hasta la ubicación del paciente. La aplicación debe mejorar el uso
del tiempo al momento de atender un paciente.
\end{onehalfspace}

La aplicación también debe automatizar la generación de reportes e
informes médicos, en base a la información recopilada en cada consulta,
también, debe presentar una vista de reportes estadísticos, para obtener
información acerca de las patologías recurrentes diagnosticadas entre
los pacientes que han sido consultados. Esta aplicación debe trabajar
en conjunto con el sistema de telemedicina y gestión de consultas
médicas vía celular (ContactaMed), que se encuentra en desarrollo,
y el sistema ya existente de citas de laboratorio (VeneLab). 

\begin{onehalfspace}
La metodología utilizada para la implementación de este proyecto es
Agile UP, una metodología de tipo ágil, con un buen balance entre
la planificación del proyecto y tiempos de desarrollo del software.\newpage{}\end{onehalfspace}
